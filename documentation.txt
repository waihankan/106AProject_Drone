Running the Aruco Node:
ros2 run ros2_aruco aruco_node //Not needed


Aruco Recognition Launch file:
ros2 launch ros2_aruco aruco_recognition.launch.py
Alternative: Launch Recognition node with TF:
ros2 launch ros2_aruco aruco_tf_launcher.launch.py

Running the webcam node:
ros2 launch usb_cam camera.launch.py


Manually set fixed frame to camera for aruco visualization

If colcon build fails:
New systems only: install colcon
Then:
rm -rf build install log
